# Glossary

## Agent

An AI system that can work toward a goal by planning, using tools, reading files, writing files, running commands, and revising its work.

## API Key

A private credential that lets a tool access an online service. Do not commit API keys to GitHub or paste them into shared files.

## Claim

A statement that needs support. In this workshop, claims may come from sources, data, code, plots, or agent-generated summaries.

## Data Claim

A statement about what a dataset shows.

## Fork

Your own GitHub copy of someone else's repository. You can make changes in your fork and propose them back with a pull request.

## Pull Request

A GitHub request to merge changes from one branch or fork into another repository.

## Source Claim

A statement about what a source says. Source claims should be checked against the original source.

## Tool Use

When an agent uses an external capability, such as reading files, writing files, running commands, searching the web, or using git.

## Verification

The process of checking whether a claim, file, script, output, or citation is actually supported.

## Workspace

The participant's working directory inside `atendee-workspace/first-last/`.
